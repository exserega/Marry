# Свадебный сайт-приглашение

Красивый и современный сайт-приглашение для свадьбы с анимацией развивающейся ткани и удобной формой регистрации гостей.

## Особенности

- 🎨 Современный дизайн с указанной цветовой палитрой
- 🌊 Анимация развивающейся ткани в шапке сайта
- 📱 Адаптивный дизайн для всех устройств
- 📝 Форма регистрации гостей
- 🎯 5 основных секций с разным содержанием
- ✨ Плавные анимации и переходы

## Цветовая палитра

- Основной: `#657362` (зеленовато-серый)
- Вторичный: `#9aa795` (светло-зеленый)
- Акцентный: `#c1c1c1` (светло-серый)
- Дополнительный: `#bd908d` (розовато-бежевый)

## Структура сайта

1. **Шапка сайта** - с анимацией развивающейся ткани и именами молодоженов
2. **Приветственная надпись** - теплое обращение к гостям
3. **Детали мероприятия** - адрес, время, дресс-код
4. **Полезная информация** - подсказки о подарках, детях, как добраться
5. **Регистрация** - форма для подтверждения присутствия

## Настройка GitHub Pages

### 1. Создание репозитория

1. Создайте новый репозиторий на GitHub
2. Назовите его `wedding-invitation` или любое другое название
3. Сделайте репозиторий публичным

### 2. Загрузка файлов

```bash
# Клонируйте репозиторий
git clone https://github.com/ваш-username/wedding-invitation.git
cd wedding-invitation

# Скопируйте все файлы проекта в папку
# (index.html, styles.css, script.js, README.md)

# Добавьте файлы в git
git add .
git commit -m "Initial commit: свадебный сайт-приглашение"
git push origin main
```

### 3. Настройка GitHub Pages

1. Перейдите в настройки репозитория (Settings)
2. Прокрутите вниз до раздела "Pages"
3. В разделе "Source" выберите "Deploy from a branch"
4. Выберите ветку "main" и папку "/ (root)"
5. Нажмите "Save"

### 4. Настройка домена (опционально)

Если у вас есть собственный домен:
1. В настройках Pages добавьте ваш домен
2. Настройте DNS записи у вашего провайдера
3. Добавьте файл `CNAME` в корень репозитория с вашим доменом

## Настройка для вашей свадьбы

### Изменение информации

1. **Имена молодоженов**: отредактируйте в `index.html` строку 15
2. **Дата свадьбы**: измените в `index.html` строку 16
3. **Место проведения**: обновите в секции "Место проведения"
4. **Время**: измените в секции "Время"

### Настройка формы

Форма регистрации настроена для демонстрации. Для реального использования:

1. Добавьте обработку на сервере (PHP, Node.js, Python)
2. Настройте отправку email уведомлений
3. Добавьте базу данных для хранения регистраций

### Изменение цветов

Цвета можно изменить в файле `styles.css` в начале файла в секции `:root`.
