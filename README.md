# 💒 Свадебный сайт - Назерке & Сергей

Современный свадебный сайт с обратным отсчетом времени и красивым дизайном.

## 🎨 Особенности

- **Обратный отсчет** до свадьбы в реальном времени
- **Большие информационные блоки** для удобного просмотра
- **Адаптивный дизайн** для всех устройств
- **Плавные анимации** и интерактивные элементы
- **Форма RSVP** для подтверждения присутствия
- **Интерактивный дресс-код** с выбором цветов

## 🎯 Версии сайта

- **index.html** - Базовая версия
- **index-v2.html** - Минималистичный дизайн
- **index-v3.html** - Романтичный стиль
- **index-v4.html** - Современный дизайн
- **index-v5.html** - Версия с обратным отсчетом (рекомендуется)

## 🚀 Как использовать

1. Откройте файл `index-v5.html` в браузере
2. Или перейдите на [GitHub Pages](https://exserega.github.io/Marry/)
